About this website

This website is built on the Angular v19 development platform. Angular is a TypeScript-based, open-source, single-page web application framework. Other technical features are:

  • Material Design using the official Angular Material components;
  • tested with the Jasmine testing framework and the Karma test runner. The Material Design components are tested with the test harness. The code test coverage is:
  • published with the Azure Static Web Apps service;
  • open-source in the my-website GitHub repository; and
  • continuous integration with GitHub Actions to manage build, test and deployment.
